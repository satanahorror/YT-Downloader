# YT Downloader

Десктопное приложение для скачивания видео и аудио с YouTube.
ffmpeg вшит в приложение — ничего дополнительно ставить не нужно.

## Быстрый старт

### 1. Установить Python (если нет)
Скачайте с https://python.org (при установке ставьте галочку "Add to PATH")

### 2. Установить зависимости
```
pip install customtkinter yt-dlp pyinstaller
```

### 3. Собрать .exe
```
python build_exe.py
```

Скрипт сам:
- Установит недостающие пакеты
- **Скачает ffmpeg** и вошьёт в приложение
- Соберёт всё в папку `dist/YT_Downloader/`

Запускайте `dist/YT_Downloader/YT_Downloader.exe` — один клик, без консоли.

## Возможности

- Скачивание видео (MP4) с выбором качества: 360p – 4K
- Скачивание аудио: MP3, M4A, OPUS, FLAC, WAV
- Скачивание плейлистов целиком
- Скачивание субтитров
- Выбор папки для сохранения
- Прогресс-бар скачивания
- Кнопка отмены
- Тёмная тема
- **ffmpeg вшит** — конвертация аудио и склейка видео работают из коробки

## Структура

```
youtube_downloader_gui.py   ← основное приложение
build_exe.py                ← скрипт сборки (скачает ffmpeg + соберёт exe)
```

## Примечание для macOS/Linux

В `build_exe.py` разделитель путей уже определяется автоматически.
ffmpeg нужно скачать вручную и положить в папку `./ffmpeg/`.
